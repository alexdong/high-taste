# Limit the number of parameters (ideally ≤ 3).

**ID**: FUNC003  
**Category**: Functions → functions

## Rule Description
Limit the number of parameters (ideally ≤ 3).

## Before
```python
def create_user(n, e, a, s):
    pass
```

## After  
```python
from dataclasses import dataclass

@dataclass
class UserProfile:
    name: str
    email: str
    age: int
    status: str

def create_user(profile: UserProfile):
    pass
```
