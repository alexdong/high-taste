# Use keyword arguments for functions with multiple boolean or numeric parameters.

**ID**: API002  
**Category**: Function and API Design â†’ functions

## Rule Description
Use keyword arguments for functions with multiple boolean or numeric parameters.

## Before
```python
connect(10, False)
```

## After  
```python
connect(timeout_ms=10, use_encryption=False)
```

## Full Example
```python
connect(use_encryption=True)
```
