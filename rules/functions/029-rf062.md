# Return Modified Value: Return the modified value rather than modifying variable in place.

## Rule Description
Return Modified Value: Return the modified value rather than modifying variable in place.

## Before
```python
def add_tax(total):
    total *= 1.2
```

## After  
```python
def add_tax(total):
    return total * 1.2
```
