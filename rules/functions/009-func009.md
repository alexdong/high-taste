# Encapsulate complex conditionals in well‑named functions or variables.

**ID**: FUNC009  
**Category**: Functions → functions

## Rule Description
Encapsulate complex conditionals in well‑named functions or variables.

## Before
```python
if (age > 65 and country == 'NZ' and insured):
    premium *= 0.8
```

## After  
```python
def eligible_for_senior_discount(age, country, insured):
    return age > 65 and country == 'NZ' and insured

if eligible_for_senior_discount(age, country, insured):
    premium *= 0.8
```
