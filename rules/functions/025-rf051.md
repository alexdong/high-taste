## Rule Description
Replace Inline Code with Function Call: Replace duplicated code with a shared function call.

## Before
```python
if now.year - birth.year > 65:
    category = 'senior'
```

## After  
```python
def age(person):
    return now.year - person.birth.year

if age(person) > 65:
    category = 'senior'
```
