## Rule Description
Substitute Algorithm: Replace old algorithm with clearer or faster one.

## Before
```python
def found_person(people):
    for p in people:
        if p == 'Don':
            return 'Don'
        if p == 'John':
            return 'John'
    return ''
```

## After  
```python
def found_person(people):
    for p in ('Don', 'John'):
        if p in people:
            return p
    return ''
```
