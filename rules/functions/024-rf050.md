## Rule Description
Replace Function with Command: Promote complex function to a command object.

## Before
```python
def save_all(records, db):
    for r in records:
        db.save(r)
```

## After  
```python
class SaveAllCommand:
    def __init__(self, records, db):
        self.records = records
        self.db = db
    def execute(self):
        for r in self.records:
            self.db.save(r)
```
