## Rule Description
Replace Temp with Query: Turn a temp variable into a query method.

## Before
```python
temp = order.quantity * order.item_price
if temp > 1000:
    return temp * 0.95
```

## After  
```python
if order.quantity * order.item_price > 1000:
    return order.quantity * order.item_price * 0.95
```
