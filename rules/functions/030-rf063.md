## Rule Description
Separate Query from Modifier: Split a method that returns data and has side effects.

## Before
```python
def get_total_and_reset(cart):
    total = sum(cart)
    cart.clear()
    return total
```

## After  
```python
def total(cart):
    return sum(cart)
def reset(cart):
    cart.clear()
```
