## Rule Description
Replace Loop with Pipeline: Use declarative pipeline (map/filter/reduce) to replace loops.

## Before
```python
names = []
for p in people:
    if p.age > 18:
        names.append(p.name.upper())
```

## After  
```python
names = [p.name.upper() for p in people if p.age > 18]
```
