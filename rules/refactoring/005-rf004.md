# Collapse Hierarchy: Remove redundant superclass/subclass when they no longer diverge.

**ID**: RF004  
**Category**: Dealing with Generalization â†’ refactoring

## Rule Description
Collapse Hierarchy: Remove redundant superclass/subclass when they no longer diverge.

## Before
```python
class ConcertTicket:
    def __init__(self, price):
        self.price = price

class MovieTicket(ConcertTicket):
    pass
```

## After  
```python
class Ticket:
    def __init__(self, price):
        self.price = price
```

## Full Example
```python
# Demonstration of Collapse Hierarchy before refactoring
```
