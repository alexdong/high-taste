# Pull Up Field: Move identical fields from subclasses to the superclass.

**ID**: RF032  
**Category**: Dealing with Generalization â†’ refactoring

## Rule Description
Pull Up Field: Move identical fields from subclasses to the superclass.

## Before
```python
class Employee:
    pass
class Engineer(Employee):
    salary = 0
class Manager(Employee):
    salary = 0
```

## After  
```python
class Employee:
    salary = 0
class Engineer(Employee):
    pass
class Manager(Employee):
    pass
```
