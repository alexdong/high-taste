# Extract Superclass: Create superclass for common behavior.

**ID**: RF016  
**Category**: Dealing with Generalization â†’ refactoring

## Rule Description
Extract Superclass: Create superclass for common behavior.

## Before
```python
class Truck:
    def __init__(self, capacity):
        self.capacity = capacity
    def fuel_efficiency(self):
        return 8

class Van:
    def __init__(self, capacity):
        self.capacity = capacity
    def fuel_efficiency(self):
        return 10
```

## After  
```python
class Vehicle:
    def __init__(self, capacity):
        self.capacity = capacity
    def fuel_efficiency(self):
        raise NotImplementedError

class Truck(Vehicle):
    def fuel_efficiency(self):
        return 8

class Van(Vehicle):
    def fuel_efficiency(self):
        return 10
```
