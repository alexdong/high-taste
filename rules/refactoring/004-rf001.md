## Rule Description
Change Function Declaration: Clarify intent by renaming a function and adjusting parameters.

## Before
```python
def circum(r):
    return 2 * 3.14159 * r
```

## After  
```python
import math

def circumference(radius: float) -> float:
    """Return the circumference of a circle."""
    return 2 * math.pi * radius
```
