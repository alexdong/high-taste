# Push Down Method: Move method used by few subclasses down to them.

**ID**: RF035  
**Category**: Dealing with Generalization â†’ refactoring

## Rule Description
Push Down Method: Move method used by few subclasses down to them.

## Before
```python
class Employee:
    def target(self):
        return 100
```

## After  
```python
class Sales(Employee):
    def target(self):
        return 100
```

