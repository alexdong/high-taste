# Use guard clauses / early returns to reduce nesting.

**ID**: CTRL001  
**Category**: Control Flow â†’ control_flow

## Rule Description
Use guard clauses / early returns to reduce nesting.

## Before
```python
def send_email(addr):
    if addr:
        if is_valid(addr):
            smtp.send(addr)
```

## After  
```python
def send_email(addr):
    if not addr or not is_valid(addr):
        return
    smtp.send(addr)
```

## Full Example
```python
# Call send_email safely
send_email(user.email)
```
