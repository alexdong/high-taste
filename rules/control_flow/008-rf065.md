## Rule Description
Split Loop: Split loop doing two things into two loops.

## Before
```python
total = 0
batch = []
for r in records:
    total += r.amount
    batch.append(transform(r))
```

## After  
```python
total = sum(r.amount for r in records)
batch = [transform(r) for r in records]
```
