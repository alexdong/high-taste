# Prefer positive conditions over negated ones for readability.

## Rule Description
Prefer positive conditions over negated ones for readability.

## Before
```python
if not error:
    proceed()
```

## After  
```python
if ok:
    proceed()
```

## Full Example
```python
ok = validate(data)
if ok:
    proceed()
```
