# Use ternary expressions only for simple assignments.

**ID**: CTRL004  
**Category**: Control Flow â†’ control_flow

## Rule Description
Use ternary expressions only for simple assignments.

## Before
```python
status = 'OK' if is_ready() else 'Pending' if queued else 'Fail'
```

## After  
```python
status = determine_status()
```

## Full Example
```python
def determine_status():
    if is_ready():
        return 'OK'
    if queued:
        return 'Pending'
    return 'Fail'
```
