# Apply De Morgan's law to simplify negated compound conditions.

**ID**: LOGIC002  
**Category**: Logic and Expressions → control_flow

## Rule Description
Apply De Morgan's law to simplify negated compound conditions.

## Before
```python
if not (ready or complete):
    wait()
```

## After  
```python
if not ready and not complete:
    wait()
```

## Full Example
```python
while not ready and not complete:
    wait()
```
