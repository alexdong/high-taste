# Replace Exception with Precheck: Check conditions before operation instead of using exceptions for control flow.

**ID**: RF049  
**Category**: Simplifying Conditional Expressions â†’ control_flow

## Rule Description
Replace Exception with Precheck: Check conditions before operation instead of using exceptions for control flow.

## Before
```python
try:
    val = cache[key]
except KeyError:
    val = fetch(key)
```

## After  
```python
if key in cache:
    val = cache[key]
else:
    val = fetch(key)
```
