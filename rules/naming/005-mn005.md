# Use searchable names (avoid single‑letter variables outside small scopes).

**ID**: MN005  
**Category**: Meaningful Names → naming

## Rule Description
Use searchable names (avoid single‑letter variables outside small scopes).

## Before
```python
k = 1024
size = bytes_read / k
```

## After  
```python
BYTES_PER_KIB = 1024
size_in_kib = bytes_read / BYTES_PER_KIB
```
