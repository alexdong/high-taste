# Make meaningful distinctions—don’t append arbitrary gradations like `data1`, `data2`.

**ID**: MN003  
**Category**: Meaningful Names → naming

## Rule Description
Make meaningful distinctions—don’t append arbitrary gradations like `data1`, `data2`.

## Before
```python
address1 = "123 Main St"
address2 = "456 Elm St"
```

## After  
```python
shipping_address = "123 Main St"
billing_address = "456 Elm St"
```
