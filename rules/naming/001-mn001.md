# Choose intention‑revealing names.

**ID**: MN001  
**Category**: Meaningful Names → naming

## Rule Description
Choose intention‑revealing names.

## Before
```python
def calc(a, b):
    return a + b
```

## After  
```python
def calculate_total_price(price, tax):
    return price + tax
```

## Full Example
```python
# Pricing utilities
```
