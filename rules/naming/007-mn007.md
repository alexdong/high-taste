# Class names are nouns; method/function names are verbs.

**ID**: MN007  
**Category**: Meaningful Names â†’ naming

## Rule Description
Class names are nouns; method/function names are verbs.

## Before
```python
class Calculate:
    pass
```

## After  
```python
class Calculator:
    def calculate(self):
        pass
```
