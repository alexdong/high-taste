# Keep functions pure—no side effects—so they are deterministic and easy to test.

**ID**: SICP015  
**Category**: Testing & Reasoning (Purity) → testing

## Rule Description
Keep functions pure—no side effects—so they are deterministic and easy to test.

## Before
```python
log = []

def add(x, y):
    log.append((x, y))
    return x + y
```

## After  
```python
def add(x, y):
    return x + y
```

## Full Example
```python
print(add(2, 3))
```
