# Tests should be independent of each other.

**ID**: TEST002  
**Category**: Unit Tests â†’ testing

## Rule Description
Tests should be independent of each other.

## Before
```python
state = {}

def test_a():
    state['x'] = 1

def test_b():
    assert state['x'] == 1
```

## After  
```python
def test_a(tmp_path):
    state = {}
    state['x'] = 1

def test_b(tmp_path):
    state = {}
    assert 'x' not in state
```
