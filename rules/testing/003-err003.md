# Avoid returning null; instead, raise exceptions or use Null Object/Optional.

**ID**: ERR003  
**Category**: Error Handling â†’ testing

## Rule Description
Avoid returning null; instead, raise exceptions or use Null Object/Optional.

## Before
```python
user = repo.find(id)
if user is None:
    return None
```

## After  
```python
try:
    return repo.get(id)
except UserNotFoundError:
    return NullUser()
```
