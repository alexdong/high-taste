# Prefer composition over inheritance.

**ID**: CLS004  
**Category**: Classes â†’ structure

## Rule Description
Prefer composition over inheritance.

## Before
```python
class TrackedList(list):
    def append(self, item):
        super().append(item)
        log(item)
```

## After  
```python
class TrackedList:
    def __init__(self):
        self._inner = []
    def append(self, item):
        self._inner.append(item)
        log(item)
```
