## Rule Description
Split Phase: Separate parsing and processing phases.

## Before
```python
rows = text.split('\n')
total = 0
for row in rows:
    cols = row.split(',')
    total += float(cols[1])
```

## After  
```python
def parse(text):
    return [row.split(',') for row in text.split('\n')]

def total_amount(rows):
    return sum(float(cols[1]) for cols in rows)

rows = parse(text)
total = total_amount(rows)
```
