## Rule Description
Classes should be small; favor a few instance variables.

## Before
```python
class Big:
    def __init__(self):
        self.a=1; self.b=2; self.c=3; self.d=4; self.e=5
```

## After  
```python
class Small:
    def __init__(self):
        self.position = (0, 0)
```
