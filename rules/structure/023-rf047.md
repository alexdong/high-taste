## Rule Description
Replace Derived Variable with Query: Remove stored redundant data and compute it when needed.

## Before
```python
class Rectangle:
    def __init__(self, width, height):
        self.width = width
        self.height = height
        self.area = width * height
```

## After  
```python
class Rectangle:
    def __init__(self, width, height):
        self.width = width
        self.height = height
    @property
    def area(self):
        return self.width * self.height
```
