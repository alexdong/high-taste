# Self-Encapsulate Field: Access a field only via its getter/setter even inside the class.

**ID**: RF013  
**Category**: Organizing Data â†’ structure

## Rule Description
Self-Encapsulate Field: Access a field only via its getter/setter even inside the class.

## Before
```python
class Thermostat:
    def __init__(self):
        self.temperature = 20
    def increase(self):
        self.temperature += 1
```

## After  
```python
class Thermostat:
    def __init__(self):
        self._temperature = 20
    def get_temperature(self):
        return self._temperature
    def set_temperature(self, value):
        self._temperature = value
    def increase(self):
        self.set_temperature(self.get_temperature() + 1)
```

## Full Example
```python
# Demonstration of Self-Encapsulate Field before refactoring
class Thermostat:
    def __init__(self):
        self.temperature = 20
    def increase(self):
        self.temperature += 1
```
