# Remove Setting Method: Make a field read‑only by removing its setter.

**ID**: RF039  
**Category**: Organizing Data → structure

## Rule Description
Remove Setting Method: Make a field read‑only by removing its setter.

## Before
```python
class Account:
    def __init__(self, id):
        self._id = id
    def set_id(self, id):
        self._id = id
```

## After  
```python
class Account:
    def __init__(self, id):
        self._id = id
    @property
    def id(self):
        return self._id
```
