# Rename Field: Rename a field to better reflect its purpose.

**ID**: RF041  
**Category**: Organizing Data → structure

## Rule Description
Rename Field: Rename a field to better reflect its purpose.

## Before
```python
class Person:
    def __init__(self):
        self.ph = '555‑1234'
```

## After  
```python
class Person:
    def __init__(self):
        self.phone_number = '555‑1234'
```

## Full Example
```python
# Demonstration of Rename Field before refactoring
```
