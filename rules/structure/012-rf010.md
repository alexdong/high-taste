## Rule Description
Encapsulate Record: Replace naked dict with a small class providing accessors.

## Before
```python
person = { 'name': 'Alice', 'age': 30 }
print(person['age'])
```

## After  
```python
class Person:
    def __init__(self, name, age):
        self._name = name
        self._age = age

    @property
    def age(self):
        return self._age

person = Person('Alice', 30)
print(person.age)
```
