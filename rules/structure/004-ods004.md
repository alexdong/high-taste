# Avoid exposing unnecessary getters/setters.

**ID**: ODS004  
**Category**: Objects and Data Structures → structure

## Rule Description
Avoid exposing unnecessary getters/setters.

## Before
```python
value = user.get_balance()
```

## After  
```python
value = user.balance  # read‑only property or via method that has meaning
```
