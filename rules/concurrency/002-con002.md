# Prefer immutability to reduce locking.

**ID**: CON002  
**Category**: Concurrency → concurrency

## Rule Description
Prefer immutability to reduce locking.

## Before
```python
shared = []

def add(x):
    shared.append(x)
```

## After  
```python
from queue import Queue
shared = Queue()

def add(x):
    shared.put(x)
```
